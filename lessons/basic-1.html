PCの基本操作とショートカット

基礎 1 / 5 ・ 目安 12分
🎬 動画は準備中です この教材は近日中に解説動画を追加予定です

在宅ワークの多くは、パソコンの中にあるファイルやアプリを使って進んでいきます。難しい専門知識は不要ですが、「よく使う操作」を先に整理しておくと、この先の学習も仕事も驚くほどスムーズになります。今日はその土台づくりです。

ファイルとフォルダを整理する

まず意識したいのは「どこに何を置いたか、自分がすぐ分かる状態」を作ることです。デスクトップにファイルを置きっぱなしにしていると、後で必ず探す時間が発生します。次のようなルールを最初に決めておきましょう。

  • 案件・仕事ごとにフォルダを分ける（例：「2026_A社_バナー制作」）
  • ファイル名の先頭に日付を入れる（例：「20260706_提案書.docx」）
  • 「とりあえずフォルダ」を1つだけ作り、迷ったら一旦そこに入れる

まず覚えたい基本のショートカット

マウスの右クリックだけでも作業はできますが、ショートカットキーを使うと同じ作業が半分以下の時間で終わります。焦らず、まずは次の基本セットから体に覚えさせましょう。

  • Ctrl + C／Ctrl + V：コピーと貼り付け
  • Ctrl + X：切り取り
  • Ctrl + Z：ひとつ前の操作に戻す（失敗しても慌てなくてOK）
  • Ctrl + Y：戻した操作をやり直す
  • Ctrl + S：上書き保存（こまめに押す癖をつける）
  • Ctrl + A：すべて選択
  • Ctrl + F：ページ内・ファイル内を検索
  • Ctrl + P：印刷・PDF書き出し

ウィンドウ・アプリ操作を早くする

アプリをいくつも開いて作業するようになると、切り替えの速さが効率を左右します。この4つを覚えるだけで、マウスに手を伸ばす回数がぐっと減ります。

  • Alt + Tab：開いているアプリを切り替える
  • Ctrl + W：今開いているタブ・ウィンドウを閉じる
  • Ctrl + T：ブラウザで新しいタブを開く
  • Windows + D：すべてのウィンドウを一度に最小化してデスクトップを表示

画面まわりの便利ワザ

知っておくと地味に助かるのが、スクリーンショットとクリップボード履歴です。クライアントに画面を見せて説明したいときなどに大活躍します。

  • Windows + Shift + S：好きな範囲だけを画像として切り取る（Macは Cmd + Shift + 4）
  • Windows + V：過去にコピーした内容を一覧から選んで貼り付ける
  • F2：選択したファイルの名前を変更する

文字入力を少し早くする

日本語入力では、変換候補が思い通りにならずイライラすることがあります。変換前のひらがな状態で「F7」を押すとカタカナに、「F10」を押すと半角英数字に一発で変換できます。この2つを覚えるだけでも、入力のストレスがかなり減ります。

POINTショートカットは、いきなり全部覚える必要はありません。次にパソコンを開いたとき、ひとつだけ意識して使ってみてください。「これ便利だな」と感じたものから自然に手が覚えていきます。
