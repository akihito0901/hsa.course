もっと便利なショートカット・画面操作

基礎 4 / 11 ・ 目安 9分
🎬 動画は準備中です この教材は近日中に解説動画を追加予定です

基本のショートカットに慣れてきたら、次は「知っていると一気に差がつく」便利ワザを覚えましょう。特に画面の切り替えとスクリーンショットは、在宅ワークで毎日使う場面が出てきます。ここで紹介するものも、全部を一度に覚える必要はありません。「便利そう」と思ったものから1つずつ試してください。

この教材のゴールは次の3つです。

  • アプリやタブを、マウスなしでサッと切り替えられる
  • 画面の好きな部分をスクリーンショットで画像にできる
  • コピー履歴と画面2分割という「時短の飛び道具」を知る

ウィンドウ・アプリを素早く切り替える

調べものをしながら文章を書く、AIに聞きながら作業する——在宅ワークでは、複数のアプリを行ったり来たりする時間が意外と長いのです。この操作を知っているとマウスに手を伸ばす回数がぐっと減ります。

  • Alt + Tab：開いているアプリを切り替える。Altを押したままTabを何度か押すと、一覧から選べます（Macは ⌘ + Tab）
  • Ctrl + T：ブラウザで新しいタブを開く
  • Ctrl + W：今のタブ・ウィンドウを閉じる
  • Ctrl + Shift + T：間違えて閉じたタブを復活させる。「あっ、消しちゃった！」の救世主です

スクリーンショット（画面の写真）を撮る

画面の一部を画像として保存できる機能です。「エラーが出たので見てほしい」「デザインのこの部分を直したい」など、言葉で説明しにくいことを一発で伝えられるので、クライアントとのやり取りで大活躍します。Windowsでの手順はこうです。

  1. 「Windowsキー + Shift + S」を押す（画面が少し暗くなります）
  2. 撮りたい範囲を、マウスでドラッグして囲む
  3. 撮った画像はコピーされた状態になるので、チャットやメールの入力欄で Ctrl + V を押して貼り付ける
  4. 画像として保存したいときは、撮った直後に画面右下に出る通知をクリックし、保存ボタンを押す

Macの場合は「⌘ + Shift + 4」で範囲を選ぶと、デスクトップに画像ファイルとして自動保存されます。

コピーの履歴を呼び出す

Windowsなら「Windowsキー + V」で、過去にコピーした内容を一覧から選んで貼り付けられます。初めて押したときは「有効にする」ボタンが出るので、押しておきましょう。「さっきコピーしたのに、別のをコピーしたら消えちゃった」という悲劇がなくなる、地味だけど強力な機能です。

画面を左右に2分割する

Windowsで「Windowsキー + ←」または「Windowsキー + →」を押すと、今のウィンドウが画面の半分にピタッと収まります。左に資料、右に書きかけの文書、と並べれば、切り替えすら不要になります。「見ながら書く」作業で一番効くワザです。

よくあるつまずきと対処法

  • Windowsキーってどれ？ → キーボード左下、Ctrlの近くにある「田」のような旗のマークのキーです
  • スクリーンショットがどこに保存されたか分からない → 「Win + Shift + S」はまずコピーされるだけです。貼り付けて使うか、直後の通知から保存します
  • Alt + Tabで目が回る → Altを押したまま、ゆっくりTabを1回ずつ押せば、順番に選べます
やってみよう今この画面の一部をスクリーンショットで撮って、自分あてのメールやLINEのキープメモに貼り付けてみましょう。「撮る → 貼る」まで通しでできたら合格です。
POINTこれらは「便利だな」と思ったものから1つずつでOK。特にスクリーンショットは、この先のコースで質問をするときにも使うので、今日必ず一度試しておきましょう。
