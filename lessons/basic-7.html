Googleドライブでファイルを管理する

基礎 7 / 11 ・ 目安 9分
🎬 動画は準備中です この教材は近日中に解説動画を追加予定です

ファイルの保存先は、パソコン本体だけでなく「Googleドライブ」を使うのがおすすめです。インターネット上（クラウド）に保存できるので、パソコンが壊れてもデータが消えず、他の人ともかんたんに共有できます。在宅ワークでは「納品物をクラウドで渡す」のが当たり前になっているので、ここでしっかり慣れておきましょう。

この教材のゴールは次の3つです。

  • Googleドライブにフォルダを作り、ファイルを保存できる
  • 共有リンクを作って、人にファイルを渡せる
  • 「閲覧のみ」と「編集可」を使い分けられる

Googleドライブとは

Googleドライブは、無料で15GBまで使えるクラウドの保管庫です。前の章で作ったGoogleアカウントがあれば、すぐに使えます。15GBというのは、文書や画像なら数千ファイルは入る容量なので、当分は足りなくなる心配はありません。似たサービスにDropboxやOneDriveもありますが、まずは1つに絞った方が迷わないので、Googleドライブから始めましょう。

クラウドに置くメリット

  • パソコンが壊れても、データはネット上に無事残っている
  • スマホからも同じファイルを見られる・編集できる
  • リンクを送るだけで、どんなに大きいファイルでも相手に渡せる（メール添付の容量制限に悩まされません）

基本の使い方

操作はとてもシンプルです。まずはフォルダを作って、ファイルを1つ入れるところまでやってみましょう。

  1. ブラウザで drive.google.com を開く（Googleアカウントでログインした状態で）
  2. 左上の「＋ 新規」ボタンを押し、「新しいフォルダ」を選んで名前を付ける
  3. フォルダを開き、パソコンのファイルを画面にドラッグ（つまんで運ぶ）すると、アップロードされる

フォルダの分け方は、基礎2で作った「仕事 → 案件ごと」のルールをそのまま持ち込めばOKです。パソコンの中とドライブの中で同じ構成にしておくと、迷子になりません。

共有リンクの作り方と注意点

ファイルやフォルダを他の人に渡すときは、リンクを作って送ります。

  1. 渡したいファイル（またはフォルダ）を右クリックし、「共有」を選ぶ
  2. 「アクセスできるユーザー」を「制限付き」から「リンクを知っている全員」に変える
  3. 右側で「閲覧者（見るだけ）」か「編集者（書き換えOK）」を選ぶ
  4. 「リンクをコピー」を押して、チャットやメールに貼り付けて送る

使い分けの基本は、「納品物や見せるだけのものは閲覧者、一緒に作業するものだけ編集者」です。案件ごとにフォルダを分けておけば、フォルダのリンク1本で全納品物を渡せてスマートです。

よくあるつまずきと対処法

  • 相手から「開けない」と言われた → 共有設定が「制限付き」のままの可能性大。「リンクを知っている全員」になっているか確認しましょう
  • アップロードが終わらない → 大きいファイルは時間がかかります。画面を閉じずに待ちましょう
  • どこに保存したか分からなくなった → 画面上部の検索窓にファイル名の一部を入れれば、すぐ見つかります
やってみようドライブに「仕事」フォルダを作り、何かファイルを1つアップロードして、その共有リンク（閲覧のみ）を作ってみましょう。作ったリンクを自分のスマホに送って開ければ、共有の流れは完璧です。
